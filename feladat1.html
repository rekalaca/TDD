A két tesztelési feladat megoldása

A projektmunka része két tetszőlegesen választott tesztelési feladat megvalósítása. Az elsőben a testtömeg index kiszámítására szolgáló metódust teszteljük, a másikban pedig a helyes email cím formátumot ellenőrizzük le. Bónuszban készült még 2 feladat a háromszögekkel kapcsolatban. Az egyik azt határozza meg, hogy megszerkeszthető e? Míg a másik a háromszög típusát határozza meg.

Lássam a feladatot!

1. feladat

Testtömegindex (BMI) számítás

A BMI (testtömegindex, Body Mass Index) egy mérőszám, amely megmutatja, hogy egy felnőtt ember sovány, normál súlyú, vagy esetleg elhízott.

(a kép forrása: wikipedia)

TDD szoftvertesztelés / Projektmunka 1.feladat:
Testtömeg-index számítása. Készítsen metódust a következő feladat megoldására a TDD szabályok betartásával! A lépések végrehajtását itt kell dokumentálnia. A dokumentációban a lépések leírása mellett ki kell fejteni, hogy mit miért és hogyan csinált! Egy projektben kell megvalósítani a feladatok megoldását és a szükséges egyéb dolgokat!

Ismerjük egy felnőtt ember magasságát és súlyát. A metódus számolja ki a testtömeg-indexét! Figyelem! A magasság 0,54m – 2,72m között lehet, a súly pedig 19,5 kg – 635 kg között elfogadható! Megjegyzés az adatokhoz: A világ valaha élt legmagasabb és legalacsonyabb embere: Robert Pershing Wadlow: 2,72 méter / Csandra Bahadur Dangi 0,54 méter Illetve a legkövérebb és legsoványabb férfija: Jon Brower Minnoch: 635 kg / Isaac Sprague: 19,5 kg

Dokumentáció lépésről lépésre:

1. A súly lehetséges értékei 19,5 – 635-ig tartó valós számok, a magasság 0,54 – 2,72 közötti valós számok.
2. Ha nem megfelelő szám érkezik a súlyra, vagy a magasságra, akkor -1 értéket adunk vissza, ha megfelelőek az értékek, akkor kg/m2 értéket.

3. Értékpárok:

  • a, mindkettő kisebb a minimumnál -> -1
  • b, magasság nagyobb, súly kisebb -> -1
  • c, mindkettő nagyobb -> -1
  • d, magasság kisebb a minimumnál, súly nagyobb -> -1
  • e, magasság kisebb, súly jó -> -1
  • f, magasság jó, súly kisebb -> -1
  • g, magasság jó, súly nagyobb -> -1
  • h, magasság nagyobb, súly jó -> -1
  • i, mindkettő érték 0 -> -1
  • j, egyik 0, másik negatív -> -1
  • k, mindkettő negatív -> -1
  • l, egyik jó, másik 0 -> -1
  • m, egyik jó, másik negatív -> -1
  • n, helyes értékpárok
4. Tesztesetek megírása a programban
5. Produkciós kód a programban
6. Minden teszt ZÖLD

IntelliJ IDEA programban készült a feladat megoldása.
Itt láthatod a teszteseteket és a produkciós kódot is: 

              
import org.junit.Test;
import org.junit.jupiter.api.DisplayName;

import static org.junit.jupiter.api.Assertions.assertEquals;

/**
 * @author rekalaca
 * @version 1.0
 */

public class TestTomegTeszt {
    /**
     * Lehetséges bemeneti értékek:
     * súly  : 19,5 – 635
     * magasság : 0,54 – 2,72
     * Ha hibás értéket adunk át paraméterként, akkor térjen vissza -1 értékkel
     * Kategóriák:
     * - minkettő negatív
     * - mindkettő nulla
     * - egyik nulla, másik negatív - 2 esetet jelent
     * - egyik nulla, másik nagyobb - 2 esetet jelent
     * - egyik jó, másik negatív - 2 esetet jelent
     * - egyik jó, másik 0 - 2 esetet jelent
     * - egyik jó, másik nagyobb - 2 esetet jelent
     * -egyik jó, másik kisebb - 2 esetet jelent
     * - mindkettő jó
     */
    @Test
    @DisplayName("Testtömeg-indexet meghatározó metódus tesztelése")
    public void tesztTomegIndex() {
        assertEquals(-1, testTomeg(0.5, 18));
        assertEquals(-1, testTomeg(3.0, 18));
        assertEquals(-1, testTomeg(3.0, 700));
        assertEquals(-1, testTomeg(0.5, 700));
        assertEquals(-1, testTomeg(0.5, 100));
        assertEquals(-1, testTomeg(1.8, 18));
        assertEquals(-1, testTomeg(1.8, 700));
        assertEquals(-1, testTomeg(3.5, 80));
        assertEquals(-1, testTomeg(0, 0));
        assertEquals(-1, testTomeg(0, -20));
        assertEquals(-1, testTomeg(-20, 0));
        assertEquals(-1, testTomeg(-1, -1));
        assertEquals(-1, testTomeg(1.8, 0));
        assertEquals(-1, testTomeg(0, 80));
        assertEquals(-1, testTomeg(-1, 100));
        assertEquals(-1, testTomeg(80, -1));
        assertEquals(25, testTomeg(2.0, 100));
        assertEquals(21.604938271604937, testTomeg(1.8, 70));
        assertEquals(26.8595041322314, testTomeg(2.2, 130));


    }

    /**
     * A testtömeg index kiszámítását végző metódus.
     * @param magassag A felnőtt ember testmagassága
     * @param suly A ferlnőtt ember testének tömege
     * @return -1, ha az értékpárok, vagy az értépárok egyik eleme az adott
     * értékhatáron kívül esik,
     * különben az eredményt adja vissza.
     */


    private double testTomeg(double magassag, double suly) {
        if (magassag < 0.54 || suly < 19.5)
            return -1;
        if (magassag > 2.72 || suly > 635)
            return -1;
        return suly / (magassag * magassag);
    }
}
