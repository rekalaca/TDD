Integrált fejlesztői környezet

Az IntelliJ IDEA egy Java IDE a JetBrains-től, elérhető Apache 2 Licenccel közösségi kiadásban és kereskedelmi kiadásban is. Gyakran egyszerűen csak "IDEA"-ként vagy "IntelliJ"-ként hivatkoznak rá. Az IntelliJ IDEA első verzióját 2001 januárjában adták ki, és akkoriban egyike volt azoknak az elsőként elérhető Java IDE-knek, amelyek továbbfejlesztett kód navigációval és kód refaktorálási képességekkel voltak felvértezve.
