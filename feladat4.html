Háromszög típus meghatározásának tesztelése

Az általános háromszög minden oldala különböző hosszú, és belső szögei is különbözőek.
Az egyenlő szárú háromszögnek legalább két oldala azonos hosszúságú. Egyben két belső szöge is ugyanakkora (az alapon fekvő szögek).
Az egyenlő oldalú háromszög, vagy szabályos háromszög minden oldala azonos hosszúságú. Egyben minden belső szöge is ugyanakkora, mégpedig 60°, szabályos sokszög.

(a kép forrása:Wikipedia)

TDD szoftvertesztelés / Projektmunka 4.feladat:

Döntsük el, hogy a szerkeszthető háromszög milyen háromszög!

Készítsen metódust a következő feladat megoldására a TDD szabályok betartásával! A lépések végrehajtását itt kell dokumentálnia. A dokumentációban a lépések leírása mellett ki kell fejtenie, hogy mit miért és hogyan csinált! Egy projektben kell megvalósítani a feladatok megoldását és a szükséges egyéb dolgokat!

Ismerjük a háromszög 3 oldalának a hosszát. A háromszög szerkeszthető, ha bármelyik két oldal hosszának az összege nagyobb, mint a harmadik oldal hossza. A háromszögek csoportosítása oldalaik hosszúsága szerint: Általános háromszög: ez egy olyan háromszög, amelynek mind a 3 oldala különböző hosszúságú. Egyenlő szárú háromszög: Ennek a háromszögnek két oldala egyenlő hosszúságú. Szabályos háromszög: Ebben a háromszögben minden oldal egyenlő hosszúságú.

Ismerjük a háromszög 3 oldalának a hosszát. A háromszög szerkeszthető, ha bármelyik két oldal hosszának az összege nagyobb, mint a harmadik oldal hossza.

Dokumentáció lépésről lépésre:

1. A háromszög oldalainak hossz értékei pozitív egész számok legyenek a feladatban.
2. Ha nem megfelelő szám érkezik akkor „nem háromszög” értékkel tér vissza, ha megfelelőek az értékek, akkor vizsgálja, hogy általános, egyenlő szárú vagy szabályos háromszög-e?
3. Értékpárok:

  • a, akármelyik értéke kisebb mint 0 ->"nem háromszög”
  • b, ha akármelyik értéke 0 -> „nem háromszög”
  • c, 3 jó érték, de nem szerkeszthető a háromszög -> "nem háromszög”
  • d, helyes értékpárok

5. Produkciós kód a programban

IntelliJ IDEA programban készült a feladat megoldása.
Itt láthatod a teszteseteket és a produkciós kódot is: 

  import org.junit.jupiter.api.DisplayName;
  import org.junit.jupiter.api.Test;
  import static org.junit.jupiter.api.Assertions.*;
  /**
   * @author koviildi
   * @version 1.0
   */
  public class ProjectHaromsz {
      @Test
      @DisplayName("Háromszög típus meghatározásának tesztelése")
      public void test_HaromszogTipus(){
          assertEquals("nem háromszög", haromszog(2,5,0));
          assertEquals("nem háromszög", haromszog(2,0,9));
          assertEquals("nem háromszög", haromszog(-1,5,8));
          assertEquals("nem háromszög", haromszog(-5,-7,-6));
          assertEquals("nem háromszög", haromszog(5,0,7));
          assertEquals("nem háromszög", haromszog(0,0,0));
          assertEquals("nem háromszög", haromszog(150,190,800));
          assertEquals("nem háromszög", haromszog(150,80,1190));
          assertEquals("nem háromszög", haromszog(-20,-20,-20));
          assertEquals("nem háromszög", haromszog(10,15,80));
          assertEquals("Általános háromszög", haromszog(15,25,30));
          assertEquals("Egyenlő szárú háromszög", haromszog(20,20,30));
          assertEquals("Egyenlő oldalú háromszög", haromszog(20,20,20));
      }
  
      /**
       * A háromszög típusának meghatározását végző metódus.
       * @param a háromszög egyik oldalának hossza,
       * @param b a háromszög második oldalának hossza,
       * @param c a háromszög harmadik oldalának hossza.
       * @return "nem háromszög" ha a háromszög két oldalának összege nem nagyobb mint
       * a harmadik oldal, vagy nulla, vagy negatív hosszúságú bármelyik oldal, különben
       * Általános, Egyenlő szárú vagy Egyenlő oldalú háromszög értékkel tér vissza.
       */
      private String haromszog(int a, int b, int c) {
          if (a+b<=c || a+c<=b || b+c<=a || a<=0 || b<=0 || c<=0)
              return "nem háromszög";
          if (a == b && b ==c)
              return "Egyenlő oldalú háromszög";
          if (a == b || a ==c)
              return "Egyenlő szárú háromszög";
          else
              return "Általános háromszög";
  
      }
  
  }
